Face Verification Project
A simple Python-based application for face verification using the DeepFace library. This project compares two images to determine if they belong to the same person and calculates the distance metric between them.

🛠 Technologies Used
Python: Core programming language.

DeepFace: A lightweight face recognition and facial attribute analysis framework.

VGG-Face: The model used for face recognition.

RetinaFace: The detector backend used for state-of-the-art face detection.

Make sure you have the required libraries installed: "pip install deepface tf-keras"
